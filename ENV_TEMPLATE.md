# Шаблон переменных окружения

## Для локальной разработки
Создайте файл `.env` в корне проекта со следующими переменными:

```env
VITE_API_URL=https://your-api-url.com
VITE_API_BASE_URL=https://your-api-base-url.com
```

## Для Timeweb APP Platform
Добавьте эти переменные в раздел "Переменные окружения" в настройках приложения на Timeweb.

**Важно:** 
- Все переменные для Vite должны начинаться с префикса `VITE_`
- В панели Timeweb переменные будут автоматически доступны во время сборки Docker образа
- НЕ добавляйте файл `.env` в git репозиторий!
