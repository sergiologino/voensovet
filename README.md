# Magic Patterns - Vite Template

## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Переменные окружения

### Клиентская часть (React/Vite)
Для настройки переменных окружения при деплое на Timeweb APP Platform см. [TIMEWEB_ENV_SETUP.md](./TIMEWEB_ENV_SETUP.md)

Для локальной разработки создайте файл `.env` в корне проекта (см. [ENV_TEMPLATE.md](./ENV_TEMPLATE.md))

### Серверная часть (Node.js)
Если у вас есть серверная часть, см. [SERVER_ENV_SETUP.md](./SERVER_ENV_SETUP.md) для настройки переменных окружения на сервере.

Для деплоя сервера на Timeweb см. [TIMEWEB_SERVER_DEPLOY.md](./TIMEWEB_SERVER_DEPLOY.md)
