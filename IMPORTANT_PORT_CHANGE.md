# ⚠️ ВАЖНО! ИЗМЕНЁН ПОРТ API

## 🔴 КРИТИЧЕСКОЕ ИЗМЕНЕНИЕ

Порт API изменён с **3001** на **3002** из-за конфликта портов при деплое.

---

## ✅ ЧТО НУЖНО СДЕЛАТЬ ПРЯМО СЕЙЧАС

### 1. Зайдите в панель Timeweb
https://timeweb.cloud/my/app-platform

### 2. Откройте Переменные окружения

### 3. ИЗМЕНИТЕ следующие переменные:

#### ❌ Старые значения (3001):
```env
VITE_API_URL=https://voensovet.ru:3001
VITE_API_BASE_URL=https://voensovet.ru:3001/api
YANDEX_REDIRECT_URI=https://voensovet.ru:3001/api/auth/yandex/callback
PORT=3001
```

#### ✅ Новые значения (3002):
```env
VITE_API_URL=https://voensovet.ru:3002
VITE_API_BASE_URL=https://voensovet.ru:3002/api
YANDEX_REDIRECT_URI=https://voensovet.ru:3002/api/auth/yandex/callback
PORT=3002
```

### 4. СОХРАНИТЕ изменения

### 5. Дождитесь автоматического редеплоя (~3-5 минут)

---

## 🔍 ПОЧЕМУ ИЗМЕНЁН ПОРТ?

**Проблема**: При деплое Timeweb не успевал остановить старый контейнер API, и новый контейнер не мог запуститься, потому что порт 3001 был занят.

**Ошибка**:
```
Bind for 0.0.0.0:3001 failed: port is already allocated
```

**Решение**: Изменён порт на 3002, чтобы избежать конфликта.

## 🆘 ЕСЛИ НЕ РАБОТАЕТ

### Проблема: Frontend не видит Backend
**Решение**: Убедитесь, что переменные `VITE_API_URL` и `VITE_API_BASE_URL` установлены на **3002**

### Проблема: Yandex OAuth не работает
**Решение**: 
1. Убедитесь, что `YANDEX_REDIRECT_URI` установлен на **3002**
2. Обновите Redirect URI в панели Yandex OAuth на `https://voensovet.ru:3002/api/auth/yandex/callback`

### Проблема: Всё ещё ошибка "port is already allocated"
**Решение**: Подождите 5-10 минут, пока Timeweb полностью остановит старые контейнеры, и попробуйте еще раз
