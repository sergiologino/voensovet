# Настройка кириллического шрифта для jsPDF

Для полной поддержки кириллицы в PDF и возможности выделения текста нужно добавить кириллический шрифт в jsPDF.

## Инструкция по добавлению шрифта:

1. **Конвертация шрифта:**
   - Откройте https://rawgit.com/MrRio/jsPDF/master/fontconverter/fontconverter.html
   - Загрузите шрифт Arial или PT Sans (TTF файл)
   - Конвертируйте его в формат jsPDF
   - Сохраните сгенерированный JavaScript файл

2. **Добавление в проект:**
   - Поместите сгенерированный файл в `src/utils/fonts/arial-normal.js`
   - Импортируйте и используйте в `ProfilePage.tsx`:

```typescript
import arialFont from './utils/fonts/arial-normal';

// В функции exportToPDF:
doc.addFileToVFS('Arial.ttf', arialFont);
doc.addFont('Arial.ttf', 'Arial', 'normal');
doc.setFont('Arial');
```

3. **Альтернативный вариант:**
   - Используйте готовый шрифт из CDN или библиотеки
   - Или используйте библиотеку pdfmake, которая уже поддерживает кириллицу

## Текущее решение:

Сейчас используется html2canvas для рендеринга HTML в изображение, что обеспечивает корректное отображение кириллицы, но не позволяет выделять текст. Для полной поддержки нужно добавить кириллический шрифт как описано выше.
